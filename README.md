# Laravel 9 Starter Kit

The package includes:
- TailwindCSS w/ DaisyUI (you can remove DaisyUI if you prefer to use plain TailwindCSS)
- AlpineJS
- ChartJS
- Simplelightbox
- Sweetalert2

## Getting started
- Clone this project.
- Run `composer install`.
- Run `npm install`.
- Run `npm run dev` or `npm run watch` to compile assets.
